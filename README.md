## Skripsi-SentimentAnalysis
Analisis Pengguna Twitter Terhadap 10 Destinasi Prioritas di Indonesia Menggunakan Metode Naive Bayes

# Menggunakan
- PHP
- Dashboard Template SB-Admin 2 (https://startbootstrap.com/themes/sb-admin-2/)
- Database MySql
